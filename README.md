# React Todo's Component

Demo of a React component.

## Installation

```bash
npm install
npm start
http://localhost:3000
```

## Technologies used

-   React
-   React Hooks
-   TypeScript
-   Local Storage

## Functionality

-   Add a todo.
-   Remove a todo.
-   Toggle todo's completion state.
-   Save todo's to local storage for persistence.

## Reason

Playing around with TypeScript inside the React.
